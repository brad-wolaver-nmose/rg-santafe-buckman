# City of Santa Fe Water Data Analysis

Multi-year water usage data analysis for the City of Santa Fe.

## Structure

- `2024/` - 2024 Buckman well analysis
- `2025/` - 2025 analysis (coming soon)
- `.claude/` - Claude Code commands shared across years

## Workflow

Each year directory contains:
- `PRD.md` - Project requirements
- `ingest_*.py` - Data ingestion scripts
- `data/` - Raw data files
- `output/` - Generated reports and CSVs

See individual year directories for specific documentation.
