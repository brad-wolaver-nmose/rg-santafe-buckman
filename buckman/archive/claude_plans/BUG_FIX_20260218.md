## Issue 1: Bug Fixes Not Committed (HIGH PRIORITY)

### Problem
Two bug fixes were applied during execution but are NOT committed to git:

**Fix 1: `step3_run_modflow.sh` (line 116)**
```bash
# Original (broken):
wine modflow96.exe "$NAM_FILE" 2>&1 | tee "${YEAR}_modflow_run.log"

# Fixed (working):
echo "$NAM_FILE" | wine ./modflow96.exe 2>&1 | tee "${YEAR}_modflow_run.log"
```
- **Reason:** MODFLOW96 expects NAM file name via stdin, not command line argument
- **Also:** Added `./` prefix for Wine to find local executable

**Fix 2: `input/modflow/2023/verify_modflow_run.py` (lines 173-175)**
```python
# Original (broken):
rates = re.findall(r"1\s+\d+\s+\d+\s+([-\d.E]+)\s+BUCKMAN", sp_section.group(1))

# Fixed (working):
# MODFLOW format: LAYER ROW COL STRESS_RATE WELL_NO
# Example:    1     13    11   -0.13733          1
rates = re.findall(r"\s+1\s+\d+\s+\d+\s+([-\d.E]+)\s+\d+", sp_section.group(1))
```
- **Reason:** MODFLOW output format has well numbers (integers), not "BUCKMAN" text

### Files

- `input/modflow/2023/verify_modflow_run.py` - Already fixed in working directory
